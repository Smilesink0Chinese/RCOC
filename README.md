# RCOC房间云通讯
### Room Chat On Cloud
房间云通讯是一个由编程猫COCO编程器创作出来的开源项目
该项目使用GNU通用开源协议，也就是GPL-3
若你使用了该项目的资源，请您将您将公开的版本进行开放源代码

由于一些特殊原因，该版本由2.0开始使用重新编写的程序

编程猫COCO编辑器尚未开放下载版本，所以编辑操作都将在浏览器进行

本人因为没有学习安卓相关的编程语言，所以使用COCO进行编写
